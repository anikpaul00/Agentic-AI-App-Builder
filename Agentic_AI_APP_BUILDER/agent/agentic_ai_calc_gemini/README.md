# Beautiful Calculator

A sleek, modern web-based calculator featuring a stunning glassmorphism design. This calculator provides a smooth user experience with both mouse and keyboard support, and supports essential arithmetic operations along with advanced UI features.

## Features

- **Arithmetic Operations**: Perform basic calculations like addition, subtraction, multiplication, and division.
- **Glassmorphism UI**: A modern, translucent interface designed with CSS backdrop filters for a premium look.
- **Dark/Light Mode**: Toggle between light and dark themes to suit your preference and environment.
- **History**: Keep track of your previous calculations with a dedicated history panel.
- **Keyboard Support**: Full keyboard integration allows you to type numbers and operators directly for faster calculations.

## Tech Stack

- **HTML5**: For the structural foundation of the application.
- **CSS3**: Leveraging modern properties like Flexbox, Grid, and Backdrop-filter for the Glassmorphism effect.
- **JavaScript (ES6+)**: Handles the calculation logic, theme switching, history management, and keyboard event listeners.

## How to Run

No installation or complex setup is required. To run the application:

1. Download or clone the repository to your local machine.
2. Navigate to the project folder.
3. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, or Edge).
