# Project Overview

A simple web‑based game that lets users play, track their scores, and reset the game. The project demonstrates clean HTML, CSS, and JavaScript integration with a focus on modularity and ease of contribution.

---

## Tech Stack

- **HTML5** – Structure of the page.
- **CSS3** – Styling and responsive layout.
- **JavaScript (ES6+)** – Game logic, score handling, and UI interactions.
- **Git** – Version control.

---

## Feature List

- Interactive gameplay with real‑time score updates.
- Persistent score display during the session.
- Reset button to start a new game.
- Responsive design for desktop and mobile browsers.
- Modular JavaScript code for easy maintenance and extension.

## Usage Guide

### How to Play
1. Load the page – the game board will appear.
2. Follow the on‑screen instructions to interact (e.g., click buttons, make selections).
3. Your actions will update the **Score** displayed at the top of the screen.

### Score Tracking
- The current score is shown in real time.
- Scores are stored in memory for the current session only (refreshing the page resets the score).

### Resetting the Game
- Click the **Reset** button to clear the current score and return the game to its initial state.
